# O que é ORM?

ORM é mapamento objeto-relacional

Consiste basicamente em mapear os dados e a estrutura do banco em objetos no projeto

## Exemplo

Tabela pessoa 
    nome , idade , sexo

class pessoa {
    String nome;
    Int idade,
    String sexo

    public save(){
        //Insert no BD
    }
}

Exemplo de ORMs

- Ruby = Rail
- Java = Hibernate
- PHP = Eloquent
- Typescript = Typeore
- Javascript = Sequelize
